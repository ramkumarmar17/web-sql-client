Usage
=====
Extract all files to a folder (WebSqlClient)
Update/Verify Connection strings in appsettings.json 
Run WebSqlClient.exe as Administrator
Access the app from Chrome: http://localhost:5000/websqlclient.html


App features
============
Add/Remove DB connections
Save frequently run SQLs (not the results)
Filter query result by column values
Download query result in csv format
Responsive layout: Accessible on Mobile/Tab/Desktop
